# ITK China 电脑信息管理系统

这是一个基于 PowerShell 和 Windows Forms 的 Windows 本地桌面工具，用来管理 ITK China 的电脑资产和归属人员。

## 当前功能

- 管理电脑名称、序列号、固定资产号、型号、MAC 地址、归属人、备注
- 电脑型号支持从已有型号下拉选择，也支持直接手动输入新型号
- 独立维护人员名单（中文名、拼音、邮箱、部门、员工类型、Mentor）
- 新增库存电脑管理界面，专门登记暂无归属同事的电脑
- 输入人员中文名时自动生成拼音简码，并允许手动修改
- 邮箱默认填充 `@itk-engineering.com`，便于继续补全
- 部门支持从已有部门下拉选择，也支持直接手动输入新部门
- 归属人输入时支持按拼音实时匹配
- 实习生必须关联一位正式员工作为 Mentor
- 编辑已有电脑记录时，保存前需要输入授权密码 `123456`
- 支持查看每台电脑的归属人历史变更记录
- 支持新增、编辑、删除、搜索电脑信息
- 支持导出 UTF-8 CSV，Excel 可直接打开
- 主界面支持 ITK Logo 展示
- 本地自动保存数据

## 数据文件

程序会在项目目录下自动维护两个文件：

- `data/computers.json`：电脑资产数据与归属历史
- `data/colleagues.json`：人员名单数据

## 启动方式

双击项目目录里的：

`启动电脑信息管理工具.bat`

或者在 Windows PowerShell 中进入项目目录后执行：

```powershell
powershell -ExecutionPolicy Bypass -File .\电脑信息管理工具.ps1
```

## 使用说明

1. 先点击“人员名单管理”，录入人员的中文名、拼音、邮箱、部门、员工类型与 Mentor 信息。
2. 输入中文名时会自动生成拼音简码，不准确时可以直接手动修改，也可以点“重新生成”。
3. 部门框会自动汇总之前录入过的部门，支持下拉选择，也支持直接输入新部门。
4. 在主界面新增或编辑电脑信息，型号框会自动汇总之前录入过的型号。
5. 如果人员类型为“实习生”，则必须选择一位正式员工作为 Mentor。
6. 在“归属人”输入框输入拼音，会自动出现匹配建议。
7. 编辑已有电脑记录时，点击“保存信息”会要求输入授权密码，当前密码为 `123456`。
8. 点击“归属历史”可以查看当前电脑的历次归属人变更记录。
9. 保存后数据会立即写入本地 JSON 文件。
10. 需要报表时，点击“导出 CSV”即可。

## 说明

- 归属人字段内部保存人员 ID，界面显示中文名。
- 删除人员前，如果该人员仍被实习生关联为 Mentor，系统会阻止删除。
- 删除人员后，其名下电脑会自动转入库存，并写入归属历史。
- Logo 文件默认读取项目目录下的 `ITK_Logo_RGB.jpg`。
- 第一版为单机本地使用，不包含多人协作或网络同步。
